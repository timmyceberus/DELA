How to build
---
1. 首先，透過下面連結內的指令先將資料庫建起來。
https://gist.github.com/benebsiny/00217e389b811ebb8e6c1733708ca7d1
2. clone專案下來：
```SHELL=
git clone https://github.com/benebsiny/DELA.git
```
3. 進入專案資料夾
```SHELL=
git npm i
git npm start
```
4. http://127.0.0.1:3000
